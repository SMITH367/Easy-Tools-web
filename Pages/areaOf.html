El objeto areaOf

El objeto areaOf, consiste en un objeto con metodos para el calculo de areas de figuras planas y algunos solidos

Metodos

areaOf.square:
Este metodo, recibe como parametro el valor del lado del un cuadrado y calcula el area en base a este aplicando la formula.

console.log(areaOf.square(4))
//La salida por consola seria:
>>> 16

areaOf.circle:
Este metodo, recibe como parametro el valor del radio de un circulo y calcula el area en base a este aplicando la formula.

console.log(areaOf.circle(6))
//La salida por consola seria:
>>> 113.097

areaOf.triangule:
Este metodo, recibe como parametros la base y la altura de un triangulo y calcula el area en base a este aplicando la formula.

console.log(areaOf.triangule(4,7))
//La salida por consola seria:
>>> 14

areaOf.rectangule:
Este metodo, recibe como parametros la base y la altura de un rectangulo y calcula el area en base a este aplicando la formula.

console.log(areaOf.rectangule(3,5))
//La salida por consola seria:
>>> 15

areaOf.rombo:
Este metodo, recibe como parametros la mayor diagonal y la menor diagonal de un rombo y calcula el area en base a este aplicando la formula.

console.log(areaOf.rombo(9,12))
//La salida por consola seria:
>>> 54

areaOf.trapecio:
Este metodo, recibe como parametros la base mayor y la base menor y la altura de un trapecio y calcula el area en base a este aplicando la formula.

console.log(areaOf.trapecio(4,7,8))
//La salida por consola seria:
>>> 32

areaOf.sphere:
Este metodo, recibe como parametro el radio de una esfera y calcula el area en base a este aplicando la formula.

console.log(areaOf.sphere(9))
//La salida por consola seria:
>>> 1017.876

areaOf.cube:
Este metodo, recibe como parametro el lado de un cubo y calcula el area en base a este aplicando la formula.

console.log(areaOf.cube(13))
//La salida por consola seria:
>>> 1014

areaOf.tetrahedron:
Este metodo, recibe como parametro la arista de un tetahedro y calcula el area en base a este aplicando la formula.

console.log(areaOf.cube(14))
//La salida por consola seria:
>>> 339.481
