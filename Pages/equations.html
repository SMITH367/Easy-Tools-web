El objeto equations

El objeto equations es una herramienta para el calculo de equaciones matematicas dentro de los numero reales de primer y segundo grado, de la forma estandar por ejemplo ax + b = c o ax2 - bx + c = 0

Metodos

equations.simple:
este metodo, recibe como parametros 3 numeros referentes a el valor de a dentro de la equacion, es decir quien tiene la x, el signo en forma de string (aritmeticos), el valor de b y el valor de c que es el que esta despues de la igualdad.

console.log(equations.simple(3,"+",8,5))
//La salida por consola seria:
>>> -1

Caso de ejemplo 2

console.log(equations.simple(9,"/",3,12))
//La salida por consola seria:
>>> 4

equations.cuadratic:
este metodo, recibe como parametros 3 numeros referentes a los valores de la equacion, el valor de a, el valor de b y el valor de c y resuelve la equacion por formula general, muestra los resultados si es que estos son numeros reales, si no muestra que el resultado no es un numero real.

console.log(equations.cuadratic(-4,12,3))
//La salida por consola seria:
>>> [ -0.2320, 3.2320 ]

Caso de ejemplo 2

console.log(equations.cuadratic(9,2,4))
//La salida por consola seria:
>>> "is not a real number"
