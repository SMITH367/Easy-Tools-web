El objeto ofArray

El objeto OfArray, consiste en un objeto con metodos para manipulacion de arreglos, con herramientas y utilidades realcionadas a esta estructura de datos en especifico

Metodos

ofArray.adition:
Este metodo, recibe como parametro un arreglo y calcula la suma de todos los elementos de este, siempre y cuando sean elementos de tipo numerico de la siguiente forma.

console.log(ofArray.adition([2,3,6]))
//La salida por consola seria:
>>> 11

ofArray.product:
Este metodo, recibe como parametro un arreglo y calcula el producto de todos los elementos de este, siempre y cuando sean elementos de tipo numerico de la siguiente forma.

console.log(ofArray.product([4,5,7]))
//La salida por consola seria:
>>> 140

ofArray.average:
Este metodo, recibe como parametro un arreglo y calcula el promedio de todos los elementos de este, siempre y cuando sean elementos de tipo numerico de la siguiente forma.

console.log(ofArray.average([2,3,4]))
//La salida por consola seria:
>>> 3

ofArray.type:
Este metodo, recibe como parametro un arreglo, evalua cada uno de los elementos y dice que tipo de arreglo retorna 6 tipos de valores en forma de string:
  • number
  • string
  • boolean
  • object
  • mixed (En caso de que uno o mas elementos de distintos tipos)
  • emply (En caso de que este vacio)

console.log(ofArray.type([2,23,90,80.8]))
//La salida por consola seria:
>>> "number"
debido a que todos los elementos son de tipo numerico

Caso de ejemplo 2

console.log(ofArray.type(["juan","as","gris"]))
//La salida por consola seria:
>>> "string"
debido a que todos los elementos son de tipo cadena

Caso de ejemplo 3

console.log(ofArray.type([54,"auto",true]))
//La salida por consola seria:
>>> "mixed"
debido a que uno o mas elementos son de distinto tipo

ofArray.arrayTObject:
Este metodo, recibe como parametro un arreglo y devuelve un objeto, del cual las key son los indices del array y los values los valores que tenia inicialmente la poscision del array

console.log(ofArray.arrayTObject(["hola mundo",231,true,9978,"txt"]))
//La salida por consola seria:
>>> {
'0': "hola mundo",
'1': 231,
'2': true,
'3': 9978,
'4': "txt"
}
