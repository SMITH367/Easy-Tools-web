El objeto generate

El objeto generate es un conjunto de metodos para generar elementos o datos aleatorios, desde numeros hasta arrays

Metodos

generate.number:
este metodo, recibe como parametro un numero minimo y un numero maximo genera un numero aleatorio dentro de ese rango.

console.log(generate.number(3,10))
//La salida por consola seria:
>>> 6

generate.string:
este metodo, recibe como parametro un numero un string con esa cantidad de caracteres.

console.log(generate.string(6))
//La salida por consola seria:
>>> "awQzcv"

generate.arrayN:
este metodo, recibe como parametro un numero y genera un array con esa n cantidad de elementos los cuales son numeros aleatorios.

console.log(generate.arrayN(4))
//La salida por consola seria:
>>> [432,756,53,2345]

generate.arrayS:
este metodo, recibe como parametro un numero y genera un array con esa n cantidad de elementos los cuales son cadenas de caracteres.

console.log(generate.arrayS(5))
//La salida por consola seria:
>>> ["Pklsdma",'a1', '2', 'MUi5ngdT', 'KdsACw']
