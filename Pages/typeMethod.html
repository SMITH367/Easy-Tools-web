La funcion: TypeMethod

Esta funcionalidad permite comprobar el tipo del valor que retorna un metodo/funcion, recibe como parametros el llamado a la funcion y el tipo esperado, retorna true si el tipo del valor retornado metodo es igual al esperado y false si no lo es.

//Definamos una funcion para sumar 2 numeros

const add = (n1,n2) => { return n1 + n2 }

Ahora apliquemos la funcion testMethod:

console.log(typeMethod(add(2,3),"number"))
//La salida por consola seria:
>>> true

Se puede tambien agregar el uno como parametro adicional (1) y hace que se devuelva un objeto con un reporte del proceso:

console.log(typeMethod(add(5,9),"string",1))
//La salida por consola seria:
>>> { expetedType: 'string', typeMethod: 'number', validation: false }
