METODOS DE EASY TOOLS JS

  • ofArray
    •

    El objeto OfArray, consiste en un objeto con metodos para manipulacion de arreglos, con herramientas y utilidades realcionadas a esta estructura de datos en especifico

  • areaOf
    •

    El objeto areaOf, consiste en un objeto con metodos para el calculo de areas de figuras planas y algunos solidos

  • perimeterOf
    •

    El objeto perimeterOf, consiste en un objeto con metodos para el calculo de perimetros de cualquier poligono y algunos solidos

  • ofConjunts
    •

    Este objeto es un conjunto de herramientas para operar con conjuntos y hacer sus respectivas operaciones, manjenado los conjuntos en forma de arreglos

  • statistic
    •

    El objeto statistic es un conjunto de elementos para calcular las distintas medidas estadisticas en base a datos ordenados los cuales se envian en forma de arraglo

  • geometryCart
    •

    El objeto geometryCart, es un conjunto de metodos para desarollar las pricipales operaciones y calculos de la geometria analitica

  • equations
    •

    El objeto equations es una herramienta para el calculo de equaciones matematicas dentro de los numero reales de primer y segundo grado, de la forma estandar por ejemplo ax + b = c o ax2 - bx + c = 0

  • formTools
    •

    El objeto forms es un conjunto de funcionalidades para validar informacion de formularios y verificar si efectivamente han sido ingresadas como el programa lo espera, contiene la gran mayoria de herramientas necesarias para esta area.

  • stack
    •

    una simulacion de la estructura de datos tipo LIFO (last input, first output) es decir el ultimo en entrar es el primero en salir, tal y como funcionan la pilas en la vida real ejemplo (una pila de libros) tambien conocida como pila, nos sirve para hacer la implementacion de esta de organizar datos

  • queue
    •

    una simulacion de la estructura de datos tipo FIFO(first input, first output) es decir el primero en entrar es el primero en salir, tal y como funcionan la colas o filas en la vida real ejemplo (fila del supermercado) tambien conocida como cola, nos sirve para hacer la implementacion de esta de organizar datos

  • linkedList
    •

    El objeto linkedList, es una simulacion de la estructura de datos "lista enlazada" es decir una forma de organizar los datos tal que esten relacionados por nodos, nos sirve para hacer la implementacion de esta forma de organizar datos

  • generate
    •

    El objeto generate es un conjunto de metodos para generar elementos o datos aleatorios, desde numeros hasta arrays

  • binarySearch
    •

    Este metodo es un algoritmo que encuentra la posicion de un valor dentro de un array ordenado aqui se maneja como una funcion, de la siguiente forma, recibe 2 parametros el valor a buscar y el array, nos retorna la poscion del elemento y si este no existe nos retorna false

  • bubbleSort
    •

    Este metodo nos a ordenar la informacion de un array con el metodo de la burbuja, es decir comparando el valor de un array con el siguiente y cambiando sus posiciones aqui se maneja como una funcion

  • quickSort
    •

    Este metodo nos a ordenar la informacion de un array de manera rapida y es el mas eficiente que existe, aqui se maneja como una funcion

  • typeMethod
    •

    Esta funcionalidad permite comprobar el tipo del valor que retorna un metodo/funcion, recibe como parametros el llamado a la funcion y el tipo esperado, retorna true si el tipo del valor retornado metodo es igual al esperado y false si no lo es.

  • functionTest
    •

    Esta funcion nos ayuda a hacer testing rapido de funciones, verificar si retonan un valor esperado y hacer unit testing a dichas funciones antes de que sean integradas, puedes testear funciones de hasta 7 parametros y lo puede hacer n veces comprobando si en todos los casos el valor es el esperado.
