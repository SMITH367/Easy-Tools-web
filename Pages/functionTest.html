La funcion functionTest

Esta funcion nos ayuda a hacer testing rapido de funciones, verificar si retonan un valor esperado y hacer unit testing a dichas funciones antes de que sean integradas, puedes testear funciones de hasta 7 parametros y lo puede hacer n veces comprobando si en todos los casos el valor es el esperado.

Aplicacion:

functionTest tiene soporte para funciones que reciban hasta 7 parametros.
Aplicacion de functionTest a funciones de 1 parametro

//Lo primero que debemos hacer es crear una funcion a la cual poder testear

const add = (n1) => { return n1 + 10 }

Para aplciar functionTest se necesita hacer el llamado y esta recibe 4 parametros, 1) el numero de parametros de la funcion a probar, el nombre de la funcion, los parametros de prueba y el restuado esperado, la funcion retornara true si el resultado de todos los casos de prueba conicide con el resultado de todos los parametros esperados, si no retornara false.

Apliquemos functionTest a esa funcion

//Lo que debemos hacer es llamar a la funcion con sus respectivos parametros

console.log(functionTest(1,add,[12],[22]))

//Lo que hacemos es enviar el numero de parametros que recibe la funcion, el nombre de esta, los parametros de prueba y los parametros esperados estos ultimos dentro de un array

//La salida por consola seria:
>>> true

Apliquemos a esta misma funcion n pruebas

console.log(functionTest(1,add,[12,1,15],[22,11,25]))

//Lo que hacemos es enviar dentro del mismo array n pruebas y n respuestas, devolvera true si todas se corresponden

//La salida por consola seria:
>>> true

Caso de ejemplo 2

console.log(functionTest(1,add,[12,10,15],[2,5,25]))

//No se cumplen todos por ende en este caso retornaria false

//La salida por consola seria:
>>> false

Aplicacion de functionTest a funciones de 2 o mas parametros

//Lo primero que debemos hacer es crear una funcion a la cual poder testear

const mult = (n1,n2) => { return n1 * n2 }

Apliquemos functionTest a esa funcion

//Lo que debemos hacer es llamar a la funcion con sus respectivos parametros

console.log(functionTest(2,mult,[[10,2]],[[20]]))

//Lo que hacemos es enviar el numero de parametros que recibe la funcion, el nombre de esta, los parametros de prueba y los parametros esperados estos ultimos dentro de un array cada una de las pruebas.

//La salida por consola seria:
>>> true

Apliquemos a esta misma funcion n pruebas

console.log(functionTest(2,mult,[[1,3],[2,4]],[3,8]))

//Lo que hacemos es enviar dentro del mismo array n pruebas y n respuestas, devolvera true si todas se corresponden

//La salida por consola seria:
>>> true

Caso de ejemplo 2

console.log(functionTest(2,mult,[[1,2],[2,4],[1,3]],[3,4,7]))

//No se cumplen todos por ende en este caso retornaria false

//La salida por consola seria:
>>> false
