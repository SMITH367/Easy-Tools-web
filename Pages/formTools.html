El objeto forms

El objeto forms es un conjunto de funcionalidades para validar informacion de formularios y verificar si efectivamente han sido ingresadas como el programa lo espera, contiene la gran mayoria de herramientas necesarias para esta area.

Metodos

forms.email:
Este metodo, recibe como parametro un string y valida si de manera general este cuenta con la estructura de un email, si es asi, retorna true, si no retornara undefined o false

console.log(forms.email("example@do.ne"))
//La salida por consola seria:
>>> true

Caso de ejemplo 2

console.log(forms.email("hola.sda"))
//La salida por consola seria:
>>> undefined

forms.phone:
El metodo form.phone, se encarga de validar si un numero telefonico ingresado tiene la longitd que el programa espera, recibe 2 numero un minimo y un maximo y el numero de telefono, retorna true si el numero esta el rango y si no retorna false

console.log(forms.phone(3,5,1241))
//La salida por consola seria:
>>> true

Caso de ejemplo 2

console.log(forms.phone(5,7,234))
//La salida por consola seria:
>>> false

forms.input:
Este metodo, se encarga de validar si un texto cumple con las condiciones deseadas por el programa, teniendo un rango de caracteres, y dando la posivilidad de prohibir caracteres, recibe 4 parametros un minimo y un maximo de caracteres, el texto en cuestion y un string con los caracteres no permitidos (opcional), retorna true si lo ingresado cumple con todas las condiciones y false si no es asi

console.log(forms.input(3,9,"hellotxt"))
//La salida por consola seria:
>>> true

Caso de ejemplo 2

console.log(forms.input(3,6,"hellotxt","x"))
//La salida por consola seria:
>>> false

forms.passaword:
Este metodo se encarga de validar si una contraseña ingresada cumple con las condiciones estandares de seguirdad, es decir que incluya al menos un numero, una letra mayuscula y una minuscula, ademas tiene una funcion para validar que la contraseña tenga una cantidad minima de caractereres (opcional), retorna true si las condiciones se cumplen y si no retorna false

console.log(forms.passaword("brayanA23"))
//La salida por consola seria:
>>> true

Caso de ejemplo 2

console.log(forms.passaword("qwa2asA",9))
//La salida por consola seria:
>>> false
