Los metodos de ordenamiento

Debido a que las estructuras de datos son utilizadas para almacenar información, para poder recuperar esa información de manera eficiente es deseable que aquella esté ordenada. Existen varios métodos para ordenar arrays. Easy Tools JS Cuenta con los metodos quickSort y bubbleSort

quickSort:

Este metodo nos a ordenar la informacion de un array de manera rapida y es el mas eficiente que existe, aqui se maneja como una funcion, de la siguiente forma

// Crearemos un array de pruebas
let array = [2,98,123,1];
// Procedemos a orderlo con quickSort
console.log(quickSort(array))
//La salida por consola seria:
>>> [1,2,98,123]


Tambien funciona para strings, su ordenamiento seria alfabetico

// Crearemos un array de pruebas
let array = ["ab","ja","cat"];
// Procedemos a orderlo con quickSort
console.log(quickSort(array))
//La salida por consola seria:
>>>[ 'ab', 'cat', 'ja' ]


bubbleSort:

Este metodo nos a ordenar la informacion de un array con el metodo de la burbuja, es decir comparando el valor de un array con el siguiente y cambiando sus posiciones aqui se maneja como una funcion, de la siguiente forma

// Crearemos un array de pruebas
let array = [2,98,123,1];

// Procedemos a orderlo con bubbleSort
console.log(quickSort(array))
//La salida por consola seria:
>>> [1,2,98,123]


Tambien funciona para strings, su ordenamiento seria alfabetico

// Crearemos un array de pruebas
let array = ["ab","ja","cat"];
// Procedemos a orderlo con bubbleSort
console.log(quickSort(array))
//La salida por consola seria:
>>>[ 'ab', 'cat', 'ja' ]

binarySearch:

Este metodo es un algoritmo que encuentra la posicion de un valor dentro de un array ordenado aqui se maneja como una funcion, de la siguiente forma, recibe 2 parametros el valor a buscar y el array, nos retorna la poscion del elemento y si este no existe nos retorna false

// Lo primero que debemos tener es un array ordenado
let array = [2,12,42,89,190];

// Procedemos a encontrar uno de sus valores con binarySearch
console.log(binarySearch(12,array))
//La salida por consola seria:
>>> 1

Caso de ejemplo 2

// Lo primero que debemos tener es un array ordenado
let array = [1,34,23,56,78];

// Procedemos a encontrar uno de sus valores con binarySearch
console.log(binarySearch(90,array))
//La salida por consola seria:
>>> false
