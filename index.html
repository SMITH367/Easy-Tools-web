Sobre Easy Tools Js Lybrary.

Easy tools nace como la idea de ser un conjunto de herramientas para la resolucion de problemas recurrentes a la hora de desarollar software, ya sea del lado del front end o del backend, brindando soluciones y funcionalidades prediseñados para que sean mas sencillos algunas tareas, En la version 1.0 de Easy Tools tenemos todo tipo de herramientas que podemos implementar dentro del lenguaje javascript, desde

  • Matematicas
  • Estructuras de datos
  • Manejo de formularios
  • Testing
  • Algortimos de uso frecuente
  • y demas herramientas utiles
claro esta que que este es un proyecto sencillo cuya idea es simplemente ser una pequeña herramienta mas, ademas se caracteriza por ser una libreria muy ligera, nuestra finalidad es brindar las herramientas escenciales para resolver problemas recurrentes.
